### 如何使用

1. **下载**

	```shell
	git clone https://github.com/CaraChen2690/Netease-Cloud-Classroom-Index.git

2. **安装**

	- Windows

	  ```shell
	  cd Netease-Cloud-Classroom-Index
	  npm install
	  ```

	- Mac

	  ```shell
	  cd Netease-Cloud-Classroom-Index
	  npm install
	  ```

3. **使用**

	a. 在命令行执行

	  ```shell
	  npm start
	  ```

	b. 在浏览器输入 [http://localhost:8080](http://localhost:8080)
